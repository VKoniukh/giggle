# Layer 1 — Laugh Topology: що система насправді моделює

> Цей документ описує **об'єкт моделювання** — не жанри гумору, а глибинні шари того, як конкретна людина переходить у стан mirth. Це фундамент, без якого будь-яка модель даних буде описувати оболонку замість суті.

---

## 7 глибинних шарів Laugh Profile

> [!IMPORTANT]
> Не "жанр / тон / довжина". Це корисні метадані, але вони описують оболонку. Потрібно розкопувати сім глибших шарів.

---

### 1. Заряджені нерви реальності (Charged Nerves)

Не просто "теми", а місця, де в людини **вже накопичена психічна енергія**:

| Нерв | Приклад |
|------|---------|
| Безсилля перед системою | Бюрократія, корпоративні ритуали |
| Власна соціальна незграбність | Ніяковість, невчасні фрази |
| Неадекватність дорослого життя | Удавання, що "все під контролем" |
| Родинна близькість і травматичність | Батьки, діти, holiday gatherings |
| Емігрантське відчуження | Подвійна ідентичність, мовні колізії |
| Розрив між образом себе і реальністю | Self-assessment vs. реальний стан |
| Секс, сором, статус | Тілесність, соціальне порівняння |
| Смерть і крихкість | Екзистенційне, лікарня, старіння |
| Інтелектуальна пиха | Самовпевненість, Dunning-Kruger |
| Повсякденне лицемірство | "Як справи?" — "Нормально" |
| Спроба зберігати гідність у хаосі | Formal composure amid collapse |

> Продукт не каже: "У тебе травма авторитету". Він просто виявляє: **коли авторитет раптом оголюється як звичайний розгублений примат — ця людина сміється.**

---

### 2. Тип перетворення (Comic Operators)

Що саме текст має **зробити** з реальністю:

```
різко переінтерпретувати
довести логіку до безумства
сприйняти метафору буквально
зіткнути два несумісні регістри
назвати приховану правду вголос
поміняти місцями сильного і слабкого
зруйнувати очікуваний punchline
дати надто серйозне пояснення повній дурниці
стиснути складний досвід в одну фразу
залишити incongruity нерозв'язаною
зробити жах буденним
зробити буденність космічно абсурдною
```

**Тема** відповідає на "про що?". **Оператор** — "що треба зробити з цим, щоб саме ця людина засміялася?"

---

### 3. Емоційне паливо сміху (Emotional Fuel)

Два тексти з однаковою структурою можуть живитися **різними емоціями**:

```
полегшення          впізнавання         ніжність
бунт                презирство          зловтіха
сором               самоприниження      спільність
відчай              агресія             безпорадне прийняття абсурду
```

> Два "чорні жарти" можуть бути психологічно протилежними: один дає жорстоке домінування, інший допомагає пережити нестерпне, третій просто шокує.

---

### 4. Дистанція і Дозвіл (Distance & Permission)

```
Надто близько  → боляче
Надто далеко   → байдуже
Надто безпечно → банально
Надто небезпечно → огидно або тривожно
```

Laugh Profile має відкрити **за якої дистанції конкретне табу стає ігровим саме для цієї людини**.

> Наприклад: людина може не переносити прямі жарти про смерть, але обожнювати бюрократичний абсурд **навколо** смерті.

> [!WARNING]
> Це значно тонша річ, ніж `dark_humor_level = 4`.

---

### 5. Голос (Voice)

Один і той самий зміст може не працювати без правильного мовця:

```
виснажений друг
наївна дитина
бюрократ
батя
надто інтелігентний невротик
людина, яка не усвідомлює власного безумства
сухий літописець катастрофи
токсичний коментатор
голос внутрішньої думки, яку соромно вимовити
```

> Гумор — це не просто "що сказано". Це **хто нібито має право так сказати**.

---

### 6. Соціальна геометрія (Social Geometry)

Кожен сміх неявно відповідає:

```
ми сміємося З КОГО?
разом ІЗ КИМ?
ПРОТИ КОГО?
хто тут СВІЙ?
хто розуміє КОД?
```

**Share** важливий не лише як "сильний лайк". Він показує: **до якої соціальної пари або групи цей жарт належить**.

"Це я відправлю колезі" і "це я відправлю дружині" — різні типи резонансу.

У перспективі Laugh Profile має знати не тільки що смішить тебе, а й **якою версією себе ти стаєш, коли смієшся з конкретною людиною**.

---

### 7. Метаболізм новизни (Novelty Metabolism)

Це **критично**. Людина може обожнювати певний нерв, але після п'яти схожих текстів він помирає.

Профіль має знати:

```
як швидко конкретний патерн насичується
скільки повторення створює "свій стиль" vs. нудоту
наскільки далеко можна відходити від знайомого
як часто потрібен wildcard
чи подобаються callbacks
чи любить людина миттєвий удар або slow burn
наскільки часто вона хоче "безумство без пояснення"
```

> Це вже не preference profile. Це **динаміка бажання**.

## 10 типів текстового Mirth

> Не жанри типу "dad joke / dark / sarcasm", а **різні стани**, які текст може викликати.

| # | Тип | Суть | Персоналізаційний потенціал |
|---|-----|------|----------------------------|
| 1 | **Сміх упізнавання** | "Це буквально я." Текст формулює досвід, якого людина ніколи не вербалізувала | 🔴 Найвищий — потребує вузької життєвої специфічності |
| 2 | **Сміх переінтерпретації** | Остання фраза змушує перебудувати все прочитане | 🟡 Середній — семантична пастка |
| 3 | **Сміх оголеної правди** | Хтось вимовляє те, що всі маскують. Не обов'язково punchline | 🔴 Високий — вимагає знання заряджених нервів |
| 4 | **Сміх абсурдної капітуляції** | Реальність перестає мати сенс. Incongruity accepted as the nature of existence | 🟡 Середній |
| 5 | **Сміх звільнення** | Дозвіл торкнутися страху, сексу, смерті, сорому без повної ваги | 🔴 Високий — потребує точної дистанції |
| 6 | **Сміх соціальної катастрофи** | Крінж. Напруга не зникає — стає смачно нестерпною | 🟡 Середній |
| 7 | **Сміх переваги** | Інший виявився дурним, фальшивим, слабким. Сильний, але легко стає токсичним | 🟢 Нижчий |
| 8 | **Сміх ніжності** | Людська недосконалість робить близькою. Спільна мила поламаність | 🔴 Високий — недооцінений пласт |
| 9 | **Мовний сміх** | Саме слово, граматика, регістр, переклад стають машиною гумору | 🟡 Середній — залежить від мовного контексту |
| 10 | **Відкладений сміх** | Спочатку дивно. Через секунду мозок добудовує зв'язок. Інша reward curve | 🟡 Середній |

## Текстові контейнери (формати подачі)

Текстовий гумор ≠ анекдотам. Анекдот — лише один контейнер.

```
діалоги                    коментарні треди
fake notifications         робочі листування
оголошення                 відгуки
пошукові запити            нотатки
інструкції                 FAQ
скарги                     confession
мікроісторії               "пост + найкращий коментар"
внутрішній монолог          абсурдний юридичний документ
fake status update         performance review excerpt
```

## Laugh Topology Summary

```
Laugh Profile = карта:
  ├── які частини реальності емоційно заряджені
  ├── які семантичні перетворення можуть зробити їх ігровими
  ├── якою мовою, голосом, інтенсивністю й дистанцією це подати
  └── як постійно повертатися до резонансу, не знищуючи його повторенням

Laugh Mining = безперервний процес:
  торкнутися
  → почути резонанс
  → зрозуміти не текст, а причину
  → мутувати причину
  → перенести в інший світ
  → схрестити з іншим нервом
  → відійти
  → здивувати
  → повернутися глибше
```

> [!NOTE]
> Оце не "персональний GPT-принтер". Це персональний алхімік, який вчиться перетворювати саме твою реальність на сміх — але ніколи не дає остаточно зрозуміти, звідки прийде наступний удар.
